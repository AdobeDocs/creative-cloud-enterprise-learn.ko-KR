# 내부 사용을 위한 메타데이터

metadata.md 파일에는 repo의 사용 안내서 TOC.md 파일로 전달되는 repo 레벨 메타데이터가 포함되어 있습니다. 사용 안내서의 metadata.md 내용을 변경하려면 TOC.md 파일에서 변경하십시오.

| 메타데이터 | 기능 |
|--- |--- |
| solution-title | 문서 헤더에서 링크로 사용됨 |
| solution-hub-url | Helpx 허브 페이지를 엽니다. |
| solution-icon | 솔루션 제목 옆에 솔루션 아이콘을 표시합니다. 아직 구현되지 않음 |
| getting-started-url | 도움말 시작 페이지 링크 |
| tutorials-url | 비디오 자습서에 연결(helpx 자습서 또는 KT 자습서) |
| mini-toc-levels | 오른쪽 레일에 나타나는 머리글 레벨 수를 결정합니다. 기본값은 2입니다. |
| git-repo | 내부 사용을 위한 마스터 리포지토리 위치를 지정합니다. |

목차 파일

| 메타데이터 | 기능 |
|--- |--- |
| user-guide-title | 문서 헤더에서 링크로 사용됨 |
| user-guide-url | Helpx 허브 페이지를 엽니다. |
