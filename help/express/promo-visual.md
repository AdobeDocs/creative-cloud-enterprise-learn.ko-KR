# 온라인 학습 강의의 프로모션 시각적 요소 만들기

온라인 학습 과정을 위한 매력적인 비주얼을 만드는 방법에 대해 알아봅니다. 수천 개에 이르는 전문적으로 디자인된 템플릿 중 하나를 리믹스하여 시작하십시오. 다음으로 사진을 추가하고 편집 기술을 사용하여 최종 비주얼의 크기를 조정하고 전달하기 전에 컴포지션에 혼합합니다.
