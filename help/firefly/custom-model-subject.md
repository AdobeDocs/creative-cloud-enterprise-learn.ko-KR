# 피사체를 사용하여 사용자 정의 모델 만들기

Firefly에서 사용자 정의 모델을 만들어 특정 주제를 사용하여 조직의 브랜드에 맞는 새로운 이미지를 만드는 방법을 알아봅니다. 맞춤형 모델을 사용하면 규모에 맞게 관련성 있고 개인 맞춤화된 콘텐츠를 제작할 수 있습니다. Adobe Firefly 사용해 보기 [여기](https://firefly.adobe.com/).
