# Adobe [!DNL Firefly] 개요

![Firefly 메인 이미지](../assets/firefly.png)

반딧불이는 Adobe 제품에 출시되는 새로운 크리에이티브 생성 AI 모델 제품군으로 초기에는 이미지 및 텍스트 효과 생성에 중점을 두고 있다. Firefoly는 크리에이티브 워크플로우를 크게 개선하면서 아이디어를 구상하고 커뮤니케이션하는 새로운 방식을 제공합니다.

<br>
