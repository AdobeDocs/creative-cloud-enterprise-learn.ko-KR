# &#x200B;4. Firefly 그래프 템플릿 라이브러리

각 템플릿은 다른 사용자가 이미 작성한 완전한 그래프입니다. 파일을 열어 정확하게 작동 방식을 확인한 다음 사용자 입력을 변경하여 원하는 버전으로 만듭니다.

| 템플릿 | 기능 | 링크 |
|---|---|---|
| **글머리 기호 VFX 생성기** | 글머리 기호 스타일 VFX 시퀀스를 생성합니다. | [템플릿 열기](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:b6edd909-614b-5c49-a79c-6d4f2cc18d03) |
| **디자인 시스템 생성기** | 재사용 가능한 디자인 시스템 에셋을 구축합니다. | [템플릿 열기](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:5300d435-e46d-5691-ba9c-501a3ba6fb25) |
| **영웅의 여정 스토리 작성기** | 스토리보드 또는 캠페인 아크에 대한 스토리 구조 및 스토리 시퀀스를 빌드합니다. | [템플릿 열기](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:065565bb-a76c-5332-ac80-9d0eed4dfdb8) |
| **사용자 지정 실시간 셰이더 및 브러시 선** | 사용자 정의 실시간 셰이더 및 브러시 획 효과를 만듭니다. | [템플릿 열기](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:aeace815-1165-5a35-a7d1-13964d3b8268) |
| **3D 턴어라운드로 스케치** | 스케치를 3D 턴어라운드 뷰로 변환합니다. | [템플릿 열기](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:27f2a0dd-42fc-53d1-9e53-2e387168a0f7) |
| **대상 색상 교정** | 비파괴적으로 선택적 지역화된 색상 교정을 적용합니다. | [템플릿 열기](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:7e5eb791-6100-503f-ab49-881da9da3e09) |
| **일관된 헤드샷** | 그룹 전체에 걸쳐 일관된 헤드샷 세트를 생성합니다. | [템플릿 열기](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:3b3bdfc4-b316-5930-ba96-44f02a99e9a4) |
| **에디토리얼 사진 촬영 + 옷 바꾸기** | 의류를 사진 촬영 시 바꿔서 에디토리얼 사진 촬영 룩을 연출합니다. | [템플릿 열기](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:c40f4d51-6579-52f0-9481-8c876e27e460) |
