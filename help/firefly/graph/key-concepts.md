# &#x200B;2. Firefly 그래프 주요 개념

Firefly 그래프를 시작하는 데 도움이 되는 주요 개념에 대해 알아보십시오.

## 노드

노드는 워크플로에서 한 단계(노드, 작업)를 수행합니다. 노드는 이미지를 생성하거나, 마스크를 적용하거나, 색상을 이동하거나, 기타 단일 크리에이티브 작업을 실행할 수 있습니다.

![노드](../assets/node.png){align="center"}

## 포트

노드의 연결점입니다. 출력 포트는 데이터를 노드 외부로 전달하고 입력 포트는 들어오는 데이터를 받습니다. 포트 연결은 워크플로우에서 데이터가 흐르는 방식입니다.

![포트](../assets/port.png){align="center"}

## 위젯

텍스트 필드, 드롭다운 및 슬라이더와 같은 노드의 대화형 컨트롤을 통해 편집기에서 직접 설정을 구성할 수 있습니다.

![위젯](../assets/widget.png){align="center"}

## Connection

연결은 두 노드 사이에서 입력 또는 출력을 전달한다. 그래프는 소스 입력에서 최종 출력까지 왼쪽에서 오른쪽으로 읽습니다.

![연결](../assets/connection.png){align="center"}

## 그래프

편집기에서 작성하는 전체 워크플로우입니다. 그래프는 최종 출력을 생성하기 위해 캔버스에 배열된 노드와 연결로 구성된다.

![그래프](../assets/graph.png){align="center"}
