# &#x200B;3. 첫 번째 그래프 만들기

노드, 연결 및 템플릿이 무엇인지 알게 되면 첫 번째 워크플로우를 빌드할 준비가 된 것입니다.

1. **빈 캔버스를 열거나 템플릿에서 시작합니다.** 노드 라이브러리 또는 템플릿 갤러리를 검색하여 필요한 항목과 가까운 시작점을 찾습니다.
2. **노드를 연결하여 워크플로우를 빌드합니다.** 노드의 출력에서 다음 노드의 입력으로 연결을 그려 체인 단계를 함께 진행합니다. 예를 들어 입력 이미지가 스타일 노드로 이동한 다음 변형 생성기로 이동한 다음 내보낼 수 있습니다.
3. **나중에 변경할 항목에 대한 입력 컨트롤을 추가합니다.** 정적 값을 슬라이더, 색상 피커 또는 텍스트 필드로 변환하여 사용자(또는 팀원)가 기본 그래프를 편집하지 않고 주요 매개 변수를 조정할 수 있도록 합니다.
4. **그래프를 실행하고 각 단계를 미리 봅니다.** 전체 워크플로우 실행이 완료되기 전에 모든 노드에서 출력을 미리 볼 수 있습니다.
5. **한 입력을 변경하고 다시 실행하십시오.** 참조 이미지를 바꾸거나, 스타일 프롬프트를 조정하거나, 변수를 수정하십시오. 다운스트림 업데이트가 모두 자동으로 업데이트됩니다. 수작업으로 재건되는 건 없어

## 먼저 시도해 볼 항목

어디서부터 시작해야 할지 잘 모르겠다면 빈 캔버스 대신 템플릿 라이브러리에서 미리 만들어진 템플릿 중 하나를 엽니다. 이는 그래프를 처음부터 만들기 전에 완전하고 작업 중인 그래프를 보는 가장 빠른 방법입니다.
